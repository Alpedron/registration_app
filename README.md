This project is a mock course registration system inspired by Oakland Community College’s Fall 2026 Software Engineering offerings.

It is a full-stack web application that allows users to manage enrollment data through CRUD (Create, Read, Update, Delete) operations across multiple related tables.
The system simulates how students enroll in courses and how data is structured in a relational database.

How to Run
1. Install XAMPP
2. Place folder inside htdocs
3. Start Apache and MySQL
4. Import database.sql into phpMyAdmin
5. Go to http://localhost/registration_app
